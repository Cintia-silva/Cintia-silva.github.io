Desenvolvedores que preferem se especializar apenas em back-end, geralmente, não são muito bons em deixar páginas bonitas e com boa usabilidade, mas são melhores em regras de negócio, banco de dados e todas as coisas que rodam no servidor. Esses desenvolvedores trabalham com linguagens de programação como, por exemplo, Java, C#, PHP, Python, Ruby ou até mesmo JavaScript (sim, tem jeito de rodar JavaScript no servidor também). Cada linguagem de programação é um mundo a parte, com comunidades, eventos, frameworks, livros, cursos, etc.
